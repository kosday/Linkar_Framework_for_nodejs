Title: Library Overview

Require:

	- Linkar.Strings/StringFunctions
	- Linkar.Functions/MvOperations
	
Set of classes to management the result of the operations Read, Update, New, Delete, Select, Dictionaries, Subroutine, Conversion, Format, Execute, LkSchemas and LkProperties.

The results of the operations with output format MV, can be used to build <LkItem> objects and list of <LkItems> objects.
With these object can build object of some <LkData> derived class, and these object can be used to management the results of the operation more friendly.
